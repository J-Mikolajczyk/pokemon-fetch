# PokeSprite Fetcher
A simple web application to fetch and display a Pokémon's sprite using the PokeAPI.

## Technologies
- HTML
- CSS
- JavaScript
- PokeAPI

## Usage
1. Enter the name of a Pokémon.
2. Click "Fetch Sprite".
3. View the Pokémon's sprite.
